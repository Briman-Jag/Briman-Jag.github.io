First Blog Entry

Topic #1: What happens when you include ‘import java.util.Scanner;’ in your code? Where does this functionality come from? What is the ‘java’ in ‘java.util.Scanner;’

The line "import java.utility.Scanner" tells the program to import the Scanner java utility package to be used in the program. Java packages contain utility classes that can be called upon with import java.util. The Scanner is a predefined class that is used to get inputs from users such as strings, ints, and doubles.

Topic #2: What is ‘void’ in a method declaration? How does this differ from declaring a method with a return type?

Void is a keyword in java used to specify that the method does not have a return type. So if it's called upon in another code it won't return anything. A method declaration with a return type like string is used to return value, like a message, in the code for the program to process further.

Topic #3: What does it mean to ‘cast a double to an int’? Give an example of when you would do this and explain what is happening.

Casting a double into an int is something that will cause an error in the program. This is because a double is a different and more complex return type than an int. Ints are whole numbers (2,1,0,-1,-2) , while doubles are floating-point numbers used for more exact calculations(2.0,1.5,0.5,-2.5). This error can happen in a program that asks the user to insert a number and the user enters a decimal number, but have it set to "keyboard.nextInt()" instead of "keyboard.nextDouble()".
